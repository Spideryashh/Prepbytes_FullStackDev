Website And Webpage

  • A webpage is a digital document.
  • It must be saved with .html extension.
  • Collection of webpage is called Website.

Types Of Website

Static

Dynamic
  • not changable
  • same content is delivered to the user.
  • faster
  • no need of database
  • created with client side languages.
  • eg: News Website, Blog Website etc.
  • changable
  • content varies with user.
  • slower than static
  • Database is required to store user data.
  • created with client side + Server side languages. eg-java,php,js etc.
  • eg: Instagram, facebook , youtube etc.

Client And Server

Client

Server
  • client is a user.
  • client request for the desired files from the server.
  • client needs browser like firefox, safari, chrome etc.
  • server is a software program.
  • it recieves clients requests & responds to it by sending the user desired files.
  • if the desired files is not present in the server then Error 404 (file not found) will be displayed.

HTML

  • stands for 'Hypertext Markup Language'.
  • developed by 'Tim Berners Lee' in '1991'.
  • created with the help of SGML(Standard Generalized Markup Language) which is complex.
  • Started with 18 Tags.
  • Published in '1995' by 'W3C'(World Wide Web Consolitation).
  • current version is 'HTML 5' released in '2014' (110+ tags).
  • case insensitive language.
  • every html documnent must be saved with .html extension.
  • without html , a browser will not know how to display the things.

Basic Tags

  1. HTML
    • it is root tag or root element.
    • every html code must start with html tag.
  2. HEAD
    • A metadata of a website or webpage should be written within head tag.
  3. BODY
    • The content to be displayed should be written within body tag.

Command Prompt

  • it is a input field in a text based user interface screen for performing some specific task.
  • cmd is case insensitive.
  • Directory : is a folder
    1. user home directory [C:,D:,E:etc]
    2. working directory [D:/user/html]

cmd commands

  1. cd : change directory
    • To move from one directory to another directory.
    • eg: cd name_of_the_folder

  2. mkdir : make directory
    • To create a directory
    • eg: mkdir name_of_the_folder

  3. cd.. : to move to previous directory

  4. cls : to clear the screen.

  5. rmdir : remove/delete directory
    • eg: rmdir name_of_the_folder

  6. dir : to list all the directories present in the current path.

  7. C: , D: : to move in different drive

  8. code . : to open selected path in vscode.

Tag

  • Keyword enclosed within angular brackets.
  • syntax : <Keyword>
  • To mark start of the tag : <Keyword>
  • To mark end of the tag : </Keyword>

Types Of Tags

Container Tag / Paired Tag Non-Container Tag / unpaired Tag/ Singleton Tag/ Singular Tag
  • needs closing
  • contain same content
  • need to write both opening and closing.
  • eg : html, head, body
  • do not need closing
  • no content
  • only need to write opening tag.
  • eg : br, img

Title Tag

  • it is a container tag.
  • used to specify or provide title to the html document.
  • it will display on the top browser tab.
  • it is written inside head tag.
  • eg : title inside head of your html shows on the tab.

HTML Element

  • Element = opening tag + content + closing tag.
  • eg : 
                        <body> 
                        I am a body tag
                    </body>

HTML Attribute

  • Attribute is the property of a tag.
  • we apply attribute so that tag can behave in user desirable manner.
  • we have to apply attribute in the opening tag.
  • eg : 
                        <body style="background-color:red"> 
                        I am a body 
                    </body>

HTML Headings

  • used to create headings in html.
  • we have 6 types of heading tag.
    h1- biggest and most important.
    h6- smallest and least important.
  • h1 should be one only, it is not considered as a good practice to use h1 more than once.
            <h1>I am heading 1.</h1>
            <h2>I am heading 2.</h2>
            <h3>I am heading 3.</h3>
            <h4>I am heading 4.</h4>
            <h5>I am heading 5.</h5>
            <h6>I am heading 6.</h6>
